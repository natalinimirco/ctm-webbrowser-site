Concrete Test Manager

WebBrowser Edition · Gestione dei getti di calcestruzzo

CTM WebBrowser è un’applicazione desktop che espone una interfaccia moderna via browser per pianificare, registrare e monitorare i getti di calcestruzzo. Tutti i dati rimangono memorizzati in locale: nessun cloud obbligatorio, nessun invio automatico all’esterno.

Scarica CTM WebBrowser (.exe)

Versione 1.0.0 · Sistema operativo: Windows 10 o successivo.

Funzioni principali

  • Archivio strutturato dei getti con cliente, cantiere, elemento e data.
  • Calcolo automatico dei giorni rimanenti e livello di urgenza.
  • Promemoria per PEC, calendario e scadenze di controllo.
  • Backup e ripristino dell’archivio dedicati all’applicazione.
  • Interfaccia via browser, con dati sempre residenti sul tuo PC.

Download e installazione

Per installare CTM WebBrowser:

  1. Clicca sul pulsante “Scarica CTM WebBrowser (.exe)” in alto.
  2. Salva il file CTM_WebBrowser_1.0_Setup.exe sul tuo PC.
  3. Esegui l’installer (potrebbe essere richiesta la conferma dell’amministratore).
  4. Al termine, avvia CTM dal collegamento sul Desktop o dal menu Start.
  5. Il browser predefinito si aprirà automaticamente con l’interfaccia di CTM.

Per maggiori dettagli operativi fai riferimento al manuale utente incluso nella directory del programma.

Informazioni e requisiti

  • Versione applicativa: CTM WebBrowser 1.0.0
  • Piattaforma: Windows 10 o successivo (64 bit consigliato).
  • Browser richiesto: Chrome, Edge, Firefox o equivalente aggiornato.
  • Connessione Internet: non necessaria per l’utilizzo ordinario.
  • Dati di lavoro e backup memorizzati in locale sul PC dell’utente.
